Gesture Sequence Recognition

Sequences are temporal patterns of existing static gesture tokens — for example, UP → DOWN → UP → DOWN = "wave". The recogniser watches the EventBus output stream and fires when a matching pattern is detected.

This tab will have its own Dataset, Train, and Config sub-panels once the sequence pipeline is implemented.
